本页是什么

这是一个静态 token 演示页，没有 JS 框架依赖，引用了 /landing/tokens.css。改了 tokens.json 跑 build 之后刷新本页即可看到效果，是设计 review 的标准入口。

详见 shared/tokens/README.md · 三端落地路径与命名约定。
