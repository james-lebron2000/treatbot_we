# 微信小程序配置完全指南

## 一、注册小程序账号

### 1. 注册流程
1. 访问 [微信公众平台](https://mp.weixin.qq.com)
2. 点击"立即注册" → 选择"小程序"
3. 填写邮箱、密码、验证码
4. 邮箱激活账号
5. 选择主体类型（个人/企业/政府/媒体/其他组织）

### 2. 主体类型对比

| 类型 | 认证费用 | 功能限制 | 适合场景 |
|------|----------|----------|----------|
| 个人 | 免费 | 不能商用、部分接口受限 | 个人开发者测试 |
| 企业 | ¥300/年 | 完整功能 | 正式商业运营 |

**建议**：Treatbot 作为医疗服务平台，建议使用**企业主体**

## 四、配置用户隐私保护指引

设置 → 基本设置 → 用户隐私保护指引

### 必填信息

**1. 收集的信息类型**：
- [x] 昵称、头像
- [x] 手机号
- [x] 位置信息（如果需要）
- [x] 相册（上传病历图片）
- [x] 摄像头（拍照上传）

**2. 信息使用目的**：
```
1. 用户身份信息用于创建个人账户
2. 病历图片用于 AI 识别和临床试验匹配
3. 联系方式用于研究机构与患者沟通
4. 位置信息用于推荐附近的临床试验机构
```

**3. 信息存储和安全**：
```
1. 所有数据存储在腾讯云服务器
2. 采用 HTTPS 加密传输
3. 敏感信息脱敏存储
4. 符合《个人信息保护法》要求
```

**4. 用户权利**：
```
1. 用户可随时查看、修改、删除个人信息
2. 用户可申请注销账户
3. 用户可撤回授权
4. 用户可导出个人数据
```

## 八、上传和发布流程

### 1. 开发版本上传

使用微信开发者工具：
1. 点击"上传"按钮
2. 填写版本号（如：1.0.0）
3. 填写项目备注
4. 确认上传

### 2. 提交审核

管理 → 版本管理 → 开发版本 → 提交审核

**审核信息填写**：
- 功能页面：pages/index/index
- 服务类目：医疗健康
- 标签：临床试验、病历管理、AI匹配

**测试账号**（必填）：
```
测试账号：test_user_001
测试密码：（如有）
```

**测试说明**：
```
1. 使用测试账号登录
2. 上传病历图片（支持拍照或相册选择）
3. 等待 AI 解析完成
4. 查看匹配结果
5. 点击报名参与试验
```

**审核备注**：
```
本小程序为临床试验信息匹配平台，帮助患者找到适合的临床试验项目。
所有试验信息来源于国家药监局公示的临床试验登记平台。
用户上传的病历仅用于 AI 分析和匹配，严格保护用户隐私。
```

### 3. 审核周期

- 普通审核：1-3 个工作日
- 加急审核：24 小时内（每月有限额）

### 4. 发布上线

审核通过后：
1. 在"审核版本"中点击"发布"
2. 选择发布模式：
   - 全量发布（所有用户可见）
   - 灰度发布（部分用户可见）
3. 确认发布

## 十一、常见问题

### Q1: 审核被拒原因？
常见原因：
- 缺少医疗资质证明
- 隐私政策不完整
- 功能不符合类目要求
- 存在敏感内容

### Q2: 如何加快审核？
- 确保功能完整可用
- 提供详细的测试说明
- 准备好相关资质证明
- 使用加急审核（有限额）

### Q3: 如何更新版本？
- 开发新版本
- 上传代码
- 提交审核
- 审核通过后发布

### Q4: 如何回滚版本？
管理 → 版本管理 → 线上版本 → 回退
